GradeBoost
------------
------------
GradeBoost is a Python project that collects data throough a questionnaire and analyzes this data in order to predict grades for 15-18 year ols students.
The project uses data from an already existing dataset as a reference to predict the students grade based on the input in the questionnaire. GradeBoost integrates
Machine Learning (a Random Forest Model) to analyze responses and provide data-driven grade forecasts.

Prerequisits
------------
- Required libraries: listed in the file requirements.txt
- GitHub API-Key: required for certain functionalities
